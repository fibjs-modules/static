# @fibjs/static

A static server written in fibjs.

## Install

```bash
$ npm i @fibjs/static -g
```

## Usage

```shell
static ./
```

## Doc

```
 Help:

 static [folder] [port]

 folder : folder of the static file to be served.
 port   : static server port. Default to 8080.
```
